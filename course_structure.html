High Level Course Modules
Module Description
Module 1: Big Data Analysis with Python and R Programming in R. Programming in Python. The programming is done in the context of a data science problem. Visulizations for data science.
Module 2: Machine Learning and AI Basics of machine learning and AI, various classification and prediction algorithms, such as, clustering, support vector machines (SVM), Naive Bayes classifier, logistic regression, regression models, deep learning, convolutinal neural network (CNN), recurrent neural network (RNN) and many more. Hands on application of these algorithms to real industry problems.
Module 3: Programming and Data Structure Programming and design of big data software architectures.
Module 4: Big Data Engineering Hands on experience on Big Data technologies such as Spark, Spark streaming, Hadoop, Kafka.
Module 5: Capstone Project A complete data science project where the students will create a model for solving a industry-based project, implement and make it work with real data. Projects are in topics such as recommendation systems, sentiment analysis, spam fitering, marketing data, prediction in market.
